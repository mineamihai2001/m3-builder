# m3-builder

## Description:

Creates a material **material design theme object** from a [**material tokens object**](https://m3.material.io/foundations/design-tokens/overview)

## Run locally:

```bash
# Get help
$ cargo run --help
```

```bash
# Default options
$ cargo run
```

```bash
# Custom options
$ cargo run --source sourceFile.json --target targetFile.json
```
